Cloud Sync

Sync your bridge data to your personal Firebase project

Sync your data to the cloud

Set up your own Firebase project to access your bridge data from anywhere. Your data stays in your Google account — no one else can see it.

Setup Your Firebase Project

1
Create a Firebase project

Go to console.firebase.google.com and click "Add project"

2
Enable Authentication

In your project: Authentication → Sign-in method → Google → Enable

3
Create Firestore Database

Firestore Database → Create database → Start in Production mode

4
Get your web app config

Project Settings → General → Your apps → Add web app → Copy the firebaseConfig object

5
Create OAuth credentials

Go to Google Cloud Console → Create Credentials → OAuth client ID (Web application) → Copy Client ID and Secret

Project:
Open Cloud Dashboard
What gets synced
  • Project names & language
  • Run status & task counts
  • Token usage & savings
  • Settings (model, supervisor)
  • File count & patterns
What is NEVER synced
  • Source code or file contents
  • Goals or task instructions
  • Diffs or code changes
  • API keys or credentials
  • File paths or error messages
Your Data

All data is stored in YOUR Firebase project. No one else — including the app developer — can access it.
