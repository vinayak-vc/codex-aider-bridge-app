AI Relay

Use any web AI as supervisor — no API key required

What is AI Relay?

Use any web-based AI (ChatGPT Plus, Claude.ai, Gemini, Grok…) as the supervisor brain — zero API keys, zero cost. You are the relay between the web AI and the bridge.

1
Generate a plan prompt
Enter your goal (e.g. "Add user authentication with JWT") and your repository path. Bridge generates a structured prompt that tells the web AI exactly what to plan.
→
2
Paste into any web AI
Open ChatGPT Plus, Claude.ai, Gemini, or any web AI. Paste the prompt. The AI will respond with a JSON task list. Copy the entire response.
→
3
Import & confirm the plan
Paste the AI's response here. Bridge parses the JSON task list and shows you every task. Review them — if something looks wrong, go back and regenerate.
→
4
Launch & review loop
Bridge executes each task using Aider (your local code editor). After each task, you get a review packet — paste it into the web AI and paste its decision back. Bridge continues automatically.
What to paste as a decision — 3 formats accepted:
✓ APPROVED
APPROVED
Task is correct. Bridge moves to the next task.
↺ REWORK
REWORK: Add input validation to the login form
Task needs changes. The text after the colon is the new instruction — be specific.
✗ FAILED
FAILED: The approach is fundamentally wrong
Task cannot be fixed. Bridge generates a replan prompt for you to paste back into the web AI.
💡 Best web AIs for planning: Claude.ai (claude-3-5-sonnet), ChatGPT-4o, and Gemini 1.5 Pro all produce clean JSON plans reliably.
⚡ Keep the same chat session open in your web AI throughout the run — it gives the AI context about what has already been done when reviewing later tasks.
🔒 Nothing leaves your machine except what you manually copy-paste. Your code, diffs, and validation output only go to the web AI when you paste the review packet.
